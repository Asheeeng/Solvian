    ⚠️ 张伟 · 矩阵行变换 04-12 16:08

    📐 错误步骤

    2
    R1 → 2·R1 行变换
    ❌
    📍 错误位置
    R2(1,1)
    ❌ 学生写的
    0
    ✅ 正确答案
    -3
    📚 知识点
    矩阵行变换
    📝 执行 R1 → 2·R1 时，学生把第二行第一个元素误改成了 0。

    🎯 错误分析

    💡 诊断说明

    执行 R1 → 2·R1 时，这个操作只对第一行生效。第二行第一个元素应保持 -3 不变，但学生误改成了 0。

    🎯
    行变换只改目标行，做完后逐行核对一遍。
    ⚠️ 王强 · 特征值求解 04-12 08:10

    📐 错误步骤

    3
    求解 det(A - λI) = 0
    ❌
    📍 错误位置
    特征值 λ
    ❌ 学生写的
    λ = 2
    ✅ 正确答案
    λ = -2
    📚 知识点
    特征值求解
    📝 展开行列式时符号判断错误，最终把特征值符号算反了。

    🎯 错误分析

    💡 诊断说明

    计算 det(A - λI) = 0 时，学生在展开行列式过程中遗漏了代数余子式的符号规则，导致结果从 -2 误写成了 2。

    🎯
    展开行列式时注意代数余子式的符号规则 (-1)^(i+j)。
    ⚠️ 李娜 · 初等行变换 04-11 19:45

    📐 错误步骤

    4
    R3 = R3 + 2R1
    ❌
    📍 错误位置
    R3 第1列、第4列
    ❌ 学生写的
    [0, 3, 2, 3]
    ✅ 正确答案
    [8, 3, 2, 9]
    📚 知识点
    初等行变换
    📝 学生只计算了部分列，没有把 2R1 的所有对应项完整加到 R3 上。

    🎯 错误分析

    💡 诊断说明

    执行 R3 = R3 + 2R1 时，学生只处理了中间两列，漏掉了第 1 列和第 4 列，因此结果没有保持整行同步更新。

    🎯
    行变换时每一列都要运算，建议列出完整计算过程并逐列核对。
